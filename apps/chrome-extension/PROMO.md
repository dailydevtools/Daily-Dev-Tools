## 📈 LinkedIn Algorithm Strategy (Pro Tips)

To get maximum reach, follow these steps:

1.  **Don't put links in the post**: Linked posts are often deprioritized by LinkedIn to keep users on the platform.
    *   **Action**: Post the text and carousel first. *Then* immediately add the first comment with the links to the Website and GitHub.
2.  **The "Dwell Time" Hack**: Carousels are the #1 way to increase "Dwell Time" (how long a user spends on your post). This is a huge signal to the algorithm. 
3.  **The Golden Hour**: Reply to **every** comment in the first 60-120 minutes. This creates a feedback loop that pushes the post to more second-degree connections.
4.  **Tag for reach**: If you have developer friends or past contributors, mention them in the comments to start the conversation.
5.  **CTA is Key**: The question at the end (*"What boring task do you Google?"*) is designed to trigger comments. If someone answers, reply with: *"That's a great one, I might add it to the roadmap. Does [Feature X] sound like it would solve that?"*
